# Change Log

All notable changes to the "opengh" extension will be documented in this file.

## [0.0.1] - 2025-11-14

### Added
- Initial release
- Open current file in GitHub/GitHub Enterprise
- Support for line selection
- Keyboard shortcut: Cmd+Shift+G Cmd+Shift+O (Mac) / Ctrl+Shift+G Ctrl+Shift+O (Windows/Linux)
